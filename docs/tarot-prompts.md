# AniMatch Tarot Card Prompts

AI image generation prompts for 47 anime characters in tarot card style.
Designed for **Flux + IP-Adapter** pipeline.

---

## Image Specs

| Property | Value |
|----------|-------|
| Resolution | 768x1024px (3:4 ratio) |
| Format | WebP (quality 85) |
| File size target | 100-200KB |
| Output path | `public/images/tarot/{heroine_id}.webp` |

## Common Style Prefix

> Prepend this to **every** character-specific prompt.

```
Anime tarot card illustration, ornate golden frame with celestial motifs, mystical background with soft bokeh lights, character centered in portrait pose, semi-realistic anime art style, detailed shading, rich jewel-tone palette, vertical composition 3:4 ratio, high detail, studio lighting
```
